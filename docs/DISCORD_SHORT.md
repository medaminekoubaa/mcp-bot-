# 🤖 MCP BOT COMMANDS - SHORT VERSION

## MCP & AI Commands

**`/ask`** - Ask MCP questions in private chat
  • Attributes: `question` (required), `context` (optional)

**`/chat`** - Create private AI chat channel
  • No attributes needed

**`/mcp-learn`** - Get MCP/Docker/AI learning materials
  • Attributes: `category` (mcp/docker/ai-llm/tools), `difficulty` (beginner/intermediate/advanced)

**`/daily-tip`** - Get educational tips
  • Attributes: `category` (mcp/ai/docker/practices)

**`/challenge-solver`** - Get help with technical problems
  • Attributes: `problem` (required), `context` (optional)

**`/ai-insights`** - Get AI recommendations
  • Attributes: `metric` (velocity/ai-usage/tools/blockers)

---

## Development Tracking

**`/dev-update`** - Log your work
  • Attributes: `project` (Career/AIRA/Other), `task_category` (feature/bug-fix/documentation/research/testing), `description` (required), `mcp_tools`, `challenges`, `status` (completed/in-progress/blocked)

**`/team-progress`** - View team stats
  • Attributes: `project` (Career/AIRA/all), `period` (today/week/month)

---

## Quick Start

1. `/chat` → Creates private channel
2. `/ask question: [your question]` → Get AI answer
3. Done! Conversation saved 💾

## 🚀 GETTING STARTED - 3 STEPS

### Step 1: Start a Private Chat Session
Type: `/chat`
→ Bot creates a private channel for you

### Step 2: Ask Your Question

### Step 3: Bot Responds
→ Get AI-powered answer about MCP/AI development
→ Conversation saved for future reference

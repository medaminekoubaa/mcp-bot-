## 🚨 Common Issues & Fixes

| Issue | Cause | Fix |
|-------|-------|-----|
| Command not showing | Not registered with Discord | Run `npm run register` |
| "Bot not responding" | Bot offline or wrong URL | Check port 3000, restart `npm start` |
| No AI insights | Groq API error | Check .env keys, check quota |
| Database error | MongoDB connection issue | Verify connection string, whitelist IP |
| System context not loaded | File missing or permissions | Check `src/prompts/system-context.txt` exists |
| Embed not showing | Missing permissions | Add "Embed Links" to bot role |
| Empty response | Null fields | Check all required options filled |
